It extract all URL's and directories for a given URL.

INSTALL:

$pip install -r requirements.txt
$sudo cp extractor.py /usr/local/bin/extractor
$chmod 755 /usr/local/bin/extractor

USAGE:

Example:

$extractor -u http://gitgub.com/ELB4rt0
